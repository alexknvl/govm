# govm

this is a blockchain system based multi-chain.
Theoretical TPS can exceed 2^64.

cn: 这是一个基于同构多链的区块链系统。它的理论性能可以超过2^64。

## Building the source

1. install golang(>v1.20),git
2. download code
3. build: go build
4. start software: ./start.sh
5. browser open http://localhost:9090
